# Healthy Hearing - CakePHP 4.x

The [Healthy Hearing](https://www.healthyhearing.com) application codebase, built with [CakePHP](https://cakephp.org) 4.x.

The framework source code can be found here: [cakephp/cakephp](https://github.com/cakephp/cakephp).

## Installation and Initial Setup
Installation and setup of our CakePHP 4.x application is accomplished by following instructions provided with the Docker-based ("devilbox") [Healthy Hearing development environment](https://github.com/Healthy-Hearing/HH-CakePHP4x-Devilbox) project that is compatible with CakePHP 4.x.
